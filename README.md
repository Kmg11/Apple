### How To Run Website On Your Maachine

- You Need To Setup [ nodejs, gulp ] on your machine
- Clone Repo
- On Terminal

```bash
  npm install
```

```bash
  npm run start
```

- Open This [Link](http://localhost:8888) On Your Browser
